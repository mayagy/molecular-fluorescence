## Abstract

This repository contains the data analysis code for a molecular fluorescence experiment. Fluorescence emission spectra were measured for three dyes (Fluorescein, Rhodamine B, Rhodamine 6G) at five concentrations each, and photographs of a laser beam attenuating through each solution were taken for Beer-Lambert analysis. The code processes spectrometer CSV data to produce emission spectra and integrated-intensity plots (Part A), and extracts beam intensity profiles from cuvette photographs to perform linear fits of ln(Intensity) vs. position (Part B). All figures used in the report are generated by these scripts.

## File Descriptions

### Scripts

- **`BL_conc.py`**
  - **Description:** Loads spectrometer emission data, cuts below 470 nm, computes log10 of intensity, numerically integrates, and plots emission spectra and integral vs. concentration for each dye.
  - **Inputs:** CSV files in `data/csv/` (one per dye-concentration combination, e.g. `F_0005.csv`).
  - **Outputs:** Emission spectra plots, per-dye integral plots, and a combined all-dyes integral plot, saved to `results/partA/`.

- **`BL_dist.py`**
  - **Description:** Reads cropped cuvette photographs, extracts the beam intensity profile along the brightest row, computes ln(Intensity) vs. position (0-10 cm), performs a weighted linear fit (Beer-Lambert), and outputs fit parameters with chi-squared statistics.
  - **Inputs:** Cropped JPEG images in `data/jpegs/` (one per dye-concentration, e.g. `R_B_005.jpg`). Images must be pre-cropped to the cuvette width.
  - **Outputs:** Per-image subdirectory in `results/` containing `heatmap.png` (single-channel heatmap), `fit.png` (linear fit with residuals), and `fit.txt` (slope, intercept, chi-squared).

- **`ods_to_csv.py`**
  - **Description:** Converts all `.ods` spectrometer files to CSV format.
  - **Inputs:** ODS files in `data/ods/`.
  - **Outputs:** CSV files in `data/csv/`.

### Data

- **`data/ods/`** — Raw spectrometer data in ODS format, one file per measurement.
- **`data/csv/`** — Converted CSV files with columns `X [nm (air)]` and `Y [Intensity]`.
- **`data/jpegs/`** — Cropped photographs of the laser beam passing through each cuvette.

File naming convention: `{dye}_{concentration}.{ext}`, where dye is `F` (Fluorescein), `R_B` (Rhodamine B), or `R_6G` (Rhodamine 6G), and concentration is in mM with a decimal point after the first digit (e.g. `01` = 0.1 mM, `0025` = 0.025 mM, `0005` = 0.005 mM).

### Results

- **`results/partA/`** — Emission spectra and integral plots from `BL_conc.py`.
- **`results/{sample}/`** — Per-image fit results from `BL_dist.py` (e.g. `results/F_01/`).
